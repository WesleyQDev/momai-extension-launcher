# Program Launcher (Extension)

This extension allows MomAI to open programs, applications, and folders directly on your computer. It is essential for automations involving opening work tools or specific directories.

### What it does for you:
*   **Open Programs**: Start any application by name (e.g., "open Chrome", "start Spotify").
*   **Find Folders**: Quickly open directories on your computer (e.g., "open my Downloads folder").
*   **Automatic Search**: It looks in common locations where your programs are usually installed so you don't have to type the full path.

### Examples of what to ask:
- "Open the Google Chrome browser."
- "Start Visual Studio Code in my documents folder."
- "Open the Pictures folder."
- "Run the calculator."

### Security:
This extension only opens items you explicitly request and respects your Windows user permissions. If it finds more than one program with a similar name, it will ask which one you want to open.
